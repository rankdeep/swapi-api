About SWAPI

The Star Wars API is the world's first quantified and programmatically-accessible data source for all the data from the Star Wars canon universe! It is also known by "swapi". Its a Star Wars Data-driven Web API to grab all the Star Wars data you want. There are data for different entities such as Characters and Planets. Apart from this, data about Species, Starships and Vehicles are also available. Moreover, data about star wars films is also available.

Statistics

  • Characters: 87
  • Planets: 61
  • Species: 37
  • Vehicles: 39
  • Starships: 37
  • Films: 7

More Info

Created in PHP along with the use of HTML and CSS. All the Data is being handled by MYSQL Database and fetched through JSON.

Published at localhost, Running through Apache Server.

This Web API is Recreated and the original StarWars API would be found at website swapi.co
